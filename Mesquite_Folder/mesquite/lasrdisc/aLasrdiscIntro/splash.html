Overview

LASRDisc (Likelihood Ancestral State Reconstruction for Discrete Characters) is a computer program to calculate the discrete, binary character states of ancestral nodes on phylogenetic trees using the maximum likelihood method. This program is designed for use only with discrete characters having two states. The Mesquite.Lasrdisc package produces the same results as the LASRDisc command-line program available from the author.

See the manual page on ancestral states for an introduction on how to use Mesquite to reconstruct ancestral states.

Installation

To install, download and unzip the file mesquite.lasrdisc.zip, placing the "lasrdisc" directory inside the Mesquite program's directory "Mesquite_Folder/mesquite"

Usage

  1. File->Open File.
  2. In Character Matrix window: Taxa&Trees->New Tree Window->Stored Tree.
  3. In Tree window: Analysis->Trace Character History.
    1. Source of Characters: Stored Characters
    2. Ancestral State Reconstruction Method:
      LASRDisc - Joint Global: [jg in LASRDisc command-line]
      All nodes are estimated simultaneously, with the parameters of the model estimated once (i.e., corresponding to Discrete's independent ancestral states global reconstruction).
      LASRDisc - Joint Local: [jl in LASRDisc command-line]
      All nodes are estimated simultaneously, with the parameters of the model estimated conditional on each state at each node (i.e., corresponding to Discrete's independent ancestral states local reconstruction).
      LASRDisc - Marginal Global: [mg in LASRDisc command-line]
      Each node is estimated independently, with the parameters of the model estimated once (i.e., corresponding to PAUP's marginal reconstruction and Discrete's fossil likelihoods global reconstruction).
      LASRDisc - Marginal Local: [ml in LASRDisc command-line]
      Each node is estimated independently, with the parameters of the model estimated conditional on each state at each node (i.e., corresponding to Discrete's fossil likelihoods local reconstruction).
    3. Source of Probability Character Models: Stored Prob. Model
    4. Probability Model: Select one.
      MK1 corresponds to S in LASRDisc command-line.
      Asymm. 2 param. corresponds to N in LASRDisc command-line.
  4. In Tree window after Trace completed: Trace->Root State Frequencies.
    Equal (0.5,0.5) [default, h in LASRDisc command-line]
    Frequency of each state at the root is equal.
    Optimized as a Free Parameter [f in LASRDisc command-line]
    Frequency of state 0 is estimated as an additional parameter to the model, state 1 is 1-ef(state 0).
    At Equilibrium Frequency [e in LASRDisc command-line]
    Equilibrium frequency of state 0 is beta/(alpha + beta), state 1 is 1-ef(state 0).
  5. When finished, Trace->Close Trace.
